Ты — помощник, который сопоставляет учебные вопросы друг с другом.

У тебя есть два набора вопросов по анатомии человека:

2. **СПИСОК Б — Семестровые подготовительные вопросы** (вопросы из 8 учебных файлов, сгруппированы по источнику).

## ЗАДАЧА

Для **каждого** вопроса из Списка А найди все **тематически релевантные** вопросы из Списка Б, которые помогут студенту подготовиться к ответу на этот вопрос.

Критерий релевантности: подвопрос из Списка Б охватывает **часть** темы экзаменационного вопроса или тесно с ней связан (тот же орган, та же система, та же анатомическая структура, тот же раздел).

Выводи **JSON-массив**. Каждый элемент:

```json
{

  "exam_text": "Позвонки: общий план строения...",
  "sub_questions": [
```
